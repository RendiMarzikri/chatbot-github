<h1>Mini Chat LLM</h1>

<h2>Fitur</h2>
<ul>
<li>Chat AI sederhana berbasis browser</li>
<li>Output kata demi kata, max 40 kata</li>
<li>Training di browser</li>
<li>Dataset bisa diedit langsung</li>
<li>Tidak perlu server</li>
</ul>

<h2>Cara Pakai</h2>
<ul>
<li>Buka <code>index.html</code></li>
<li>Isi dataset chat:</li>
</ul>
<pre>
&lt;USER&gt; halo
&lt;ASSISTANT&gt; halo juga aku adalah AI ramah
</pre>
<ul>
<li>Klik <strong>Train</strong></li>
<li>Tulis pesan di input</li>
<li>Klik <strong>Send</strong> → AI generate hingga 40 kata</li>
</ul>

<h2>Struktur</h2>
<pre>
Mini-Chat-LLM/
├─ index.html
└─ README.html
</pre>
